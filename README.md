# Advent of Code 

There are my solutions to [Advent of Code](https://adventofcode.com/) in C#. I usually get bored eventually, so its never complete.

Grab your session cookie from the AoC website and place it in a file named `AoCSessionCookie.txt` in the root of the project to enable automatic input fetching.

"Framework" provided by [AoCHelper](https://github.com/eduherminio/AoCHelper).
